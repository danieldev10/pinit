Pinit Privacy Policy

Effective date: April 27, 2026

Overview

Pinit Pinterest Downloader helps users download Pinterest video media as MP4 files. This policy explains what information the Pinit Chrome extension and hosted backend process to provide that download feature.

Information Pinit Processes

Pinit only processes information needed for its single purpose:

  • Pinterest video media URLs from the active Pinterest tab, such as pinimg.com MP4 or HLS video URLs. These URLs are sent to the Pinit backend when a user starts a download.
  • The Pinit backend URL setting, which may be stored in Chrome storage so the extension can remember which backend to use.
  • Standard request metadata that may be processed by the hosting provider, such as IP address, browser request details, timestamps, and error logs.

Information Pinit Does Not Collect

Pinit does not ask for or collect Pinterest passwords, Pinterest session cookies, authentication tokens, payment information, health information, personal communications, contact lists, or precise location data.

How Information Is Used

Pinit uses the detected video media URL to prepare and stream an MP4 download to the user's device. Request metadata may be used for security, troubleshooting, abuse prevention, and service reliability.

Sharing And Sale Of Data

Pinit does not sell user data. Pinit does not use user data for advertising, retargeting, creditworthiness, or lending purposes. Data is only transferred when necessary to provide the download feature, operate the hosted backend, comply with law, or protect the security of the service.

Retention

Pinit does not create user accounts and does not intentionally store downloaded videos. Video media is processed for the active download request. Hosting providers may retain limited server logs for operational and security purposes.

Chrome Web Store Limited Use

Pinit uses data only to provide or improve its user-facing Pinterest video download feature. Pinit does not sell user data, does not use or transfer user data for unrelated purposes, and does not use user data to determine creditworthiness or for lending.

User Choices

Users can stop Pinit from processing Pinterest video media by not clicking the Download MP4 button, disabling the extension, or uninstalling it from Chrome. Users can also reset the extension's backend URL setting from the extension popup.

Security

Pinit uses HTTPS for communication with the hosted backend. No internet service can be guaranteed to be completely secure, but Pinit is designed to process only the limited information needed for video downloads.

Changes

This privacy policy may be updated as Pinit changes. The effective date above will be updated when material changes are made.
